# Knowledge Base Folder

Drop your source files here, then click **Sync `knowledge_base`** in the UI.

Supported file types:
- PDF: `.pdf`
- Text/Docs: `.txt`, `.md`, `.markdown`, `.rst`, `.docx`
- Video metadata: `.mp4`, `.mov`, `.mkv`, `.avi`, `.webm`

For video retrieval quality, add a sidecar transcript with the same filename:
- `my_video.mp4`
- `my_video.txt` or `my_video.md`

The sync endpoint indexes everything recursively from this folder.
